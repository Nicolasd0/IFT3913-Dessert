Le programme est un jar qui s'appelle TP2.jar

Compilation:
	Rouler la commande: java -jar TP2.jar PATH_TO_FOLDER OUTPUT_PATH.CSV

	Où PATH_TO_FOLDER est le chemin su dossier root du projet jfreechart-master par exemple
	et OUTPUT_PATH.csv est le chemin du fichier retourné par le programme contenant les métriques.
	OUTPUT_PATH est un paramètre optionnel. 

Format de retour:
Le programme retourne la liste des métriques nécessaire à l'analyse fait dans le rapport sous forme de CSV.
